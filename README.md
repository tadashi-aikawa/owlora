# Owlora


## Optional

```
$ yarn global add webpack
```

## Install dependencies

```
$ yarn install
```

## Debug

```
$ yarn dev
```

## Show storybook

```
$ yarn storybook
```

## Test

```
$ yarn test
```

or

```
$ yarn test:watch
```

## Build

```
$ yarn build
```

## Deploy

1. Increment version in `package.json`
2. `yarn deploy`
